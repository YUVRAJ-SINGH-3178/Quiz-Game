# Quiz Game 🎮

This is a simple command-line quiz game built using C and C++.

## 🧠 Features
- ✅ Guess the Number
- ✅ Math Quiz
- ✅ Rock Paper Scissors
- ✅ Trivia Questions
- ✅ Even or Odd Game
- ✅ Multiplication Test

## 🛠 How to Run
Compile the code using a C++ compiler like `g++`:

```bash
g++ project1.cpp -o quizgame
./quizgame
